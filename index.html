Tecnologias de Interesse:

Front-End

Client-Side

É o desenvolvimento da interface gráfica que será visualizada e que possa ter uma interação pelo usuário de um site, sendo que é aquilo que funciona no próprio navegador do usuário, por exemplo.

HTML 5

Acrônimo de " HyperText Markup Language" (significa "Linguagem de Marcação de HiperTexto"), criada em 1991, pelo britânico Tim Berners-Lee, o qual é o criador da WWW (World Wide Web), a Web, e diretor da W3C (World Wide Web Consortium), organização que estabelece padrões para conteúdos da Web.

HTML é uma linguagem que possui Tags (Marcadores) para melhor estruturar e organizar o conteúdo, que é baseado em texto.

É por meio dessa linguagem e de toda a sua estrutura, que os navegadores interpretam o conteúdo em forma de código dos servidores e traduzem na forma visual como vemos quando acessamos um site (porém, com estilo padrão muito simples).

CSS 3

Acrônimo de "Cascading Style Sheets" (significa "Folhas de Estilo em Cascata"), proposta em 1994, pelo norueguês Håkon Wium Lie, e desenvolvido em 1995 pela W3C, é uma linguagem que determina o layout (aparência) das páginas Web.

É a estilização e design que os conteúdos de textos serão apresentados, utilizando-se das Tags HTML para ser aplicado.

JS

Acrônimo de "JavaScript", criada em 1995 pelo americano Brendan Eich para a Netscape (empresa criadora de um dos primeiros navegadores usados na Web, o Netscape Navigator, 1994, a qual também deu origem ao Mozilla, qua atualmente tem o navegador Firefox, 1998).

É uma Linguagem de Programação do tipo script (que pode ser usada para dar instruções a outros programas). No caso, ela é muito utilizada em navegadores para tornar a exibição mais dinâmica, podendo modificar a apresentação de uma página Web sem que sejam necessárias requisições ao servidor para cada interação do usuário. Com ela é possível alterar o conteúdo (adicionar e/ou remover texto e design), dar movimento em componentes e assim, permitir uma melhor interação do usuário.

Para atender o pedido da Netscape que colaborava com a Sun Microsystens (criadora da linguagem de programação JAVA) para desenvolver um navegador (que a ideia inicial era ter o JAVA complementado por uma outra linguagem de script que tivesse a mesma sintaxe do JAVA) que iria concorrer com a Microsoft (Interne Explorer), o protótipo da linguagem foi desenvolvido em 10 dias e teve o nome registrado como Mocha, mas chamada de LiveScript, o qual foi influenciado pela linguagem Scheme (de 1975, baseada em Lisp, de 1958) e mais tarde por Self (de 1986), Perl (de 1987) e Python (de 1991).

Como o JAVA era muito famoso na época, uma estratégia de marketing da Sun renomeou o LiveScript para JavaScript, para pegar carona nesta fama, porém, são linguagens diferentes e não possuem relação.

Em 1996, a Microsoft implementou em seu navegador o JScript, uma variação do JavaScript sendo uma engenharia reversa. Com o surgimento de outros navegadores, a Netscape submeteu o JavaScrip à ECMA International (acrônimo para "European Computer Manufacturers Association", que significa "Associação Europeia de Fabricantes de Computadores") para que fosse especificado uma padronização, onde outros fornecedores de navegador poderiam implementar seu uso. Essa padronização (edição de publicação da norma ECMA-262, em 1997) deu origem a um novo nome: ECMAScript.

Com o passar do tempo, o ECMAScript necessitava de atualizações, sendo que, para que fosse implementado, era necessário a aprovação de um comitê formado entre os fornecedores de navegadores.

Back-End

Server-Side

É o desenvolvimento de recursos e estrutura por trás de uma interface gráfica, uma estrutura que garante a interação do usuário com a parte gráfica e é relacionada com servidores, conectando banco de dados e recursos com processos lógicos, como a garantia da segurança.

C

Criada em 1972, pelo americano Dennis Ritchie (criador da Linguagem B, em 1969, e que também participou da criação do Multix, em 1969, e Unix, em 1969) no Bell Labs da AT&T (atualmente são subsidiária da Nokia). É uma linguagem de programação compilada e de propósito geral. Influenciada por ALGOL (de 1968, que fora influenciada por FORTRAN, de 1957).

C é uma linguagem imperativa e procedural para implementação de sistemas. Seus pontos de design foram para ele ser compilado, fornecendo acesso irrestrito à memória e baixos requerimentos do hardware. Também foi desenvolvido para ser uma linguagem de alto nível, para maior reaproveitamento do código. C foi útil para muitas aplicações que foram codificadas originalmente em Assembly.

Na época, ela permitia que programadores não ficassem tão sobrecarregados, que por ter mais simplicidade, facilitava a criação de programas extensos e com menos erros. Bem como permitia o programador permanecer sempre em controle do que o programa está fazendo.

Muito utilizado no desenvolvimento de sistemas e softwares de base, por ser muito eficiente, mas, por permitir aos programadores grande liberdade, as possibilidades de erros também são grandes.

C++

Criada em 1979, pelo dinamarquês Bjarne Stroustrup no Bell Labs da AT&T (atualmente são subsidiária da Nokia), é uma linguagem que deriva da linguagem C, sendo que o "++" (lê-se em inglês como "plus plus") é um símbolo utilizado na programação para dizer o "incremento de 1 (um)", que no caso, é em referência que o C++ seria a Linguagem C com algum incremento (evolução), sendo muito difundido atualmente, devido ao maior contraste entre elas, a Programação Orientada a Objetos (POO).

Com o objetivo implementar uma versão distribuída do núcleo Unix, e com alguns desafios que incluíam simular a infraestrutura da comunicação entre processos num sistema distribuído ou de memória compartilhada e escrever drivers para tal sistema, Stroustrup optou por acrescentar elementos de outras linguagens na linguagem C, como o Simula 67 (de 1967, baseado em ALGOL), para que pudesse ter melhor eficiência nestes trabalhos.

Inicialmente, chamado de "C With Classes" (traduzido: C com Classes), teve seu nome alterado para C++ em 1983. Passou a ter um compilador próprio e também foi incrementando novas características, as quais passaram a fazer com que a C++ se diferenciasse da C, sendo que em 1989, sua segunda versão, foi também incrementado o suporte a orientação a objetos.

Stroutrup citou: "C faz com que dar um tiro no pé seja fácil; C++ torna isso mais difícil, mas quando nós o fazemos arrebentamos com a perna toda."

C++ é uma linguagem de programação Open Source (Código Aberto) orientada a objetos, utilizada tanto como linguagem de máquina quanto para softwares empresariais, acadêmicos, jogos, edição de imagens, fotos e vídeos. Tal popularidade fez com que influenciasse muitas outras linguagens.
